# РКН Симулятор: Депутат 2026

Сатирическая браузерная игра: балансируйте бюджет, доверие граждан и «очистку сети», принимайте законы и переживайте случайные события. Это **не** реальная политика — юмор и абсурд.

## Запуск

Статическая игра: достаточно открыть `index.html` в браузере. Для **фоновой музыки** и **чит-меню** (Web Crypto API) лучше открыть через HTTP:

```bash
npm start
```

Откройте в браузере адрес, который покажет сервер (обычно `http://localhost:3000`).

Альтернатива без Node: `python -m http.server 8080` в папке проекта и переход на `http://localhost:8080`.

## Сборка для публикации

Загрузите на GitHub Pages, Netlify, Cloudflare Pages или любой хостинг статики: `index.html`, `Vladimir_Putin_-_Molodec_politik_lider_i_borec_72438381.mp3`, `Slava_KPSS_Pyrokinesis_-_RKN_74532024.mp3`, `dev-secrets.js`, `dev-secrets.example.js`, `LICENSE`, при желании `package.json` и этот `README.md`. В `dev-secrets.js` по умолчанию только комментарии — свой хэш для чит-меню можно задать локально по подсказкам в файле.
